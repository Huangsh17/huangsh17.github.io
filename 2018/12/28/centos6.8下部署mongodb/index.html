centos6.8下部署mongodb

发表于 2018-12-28 | 分类于 数据库

安装前准备

最新mongodb下载链接查找方式
进入官网：https://www.mongodb.com/download-center?jmp=nav#community
点击 Community Server，再点击 linux
version(版本)选择 Linux 64-bit legacy x64
然后DOWNLOAD下载按钮右边有个下载链，https://***，如下
https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-4.0.2.tgz

一、安装

进入home目录，我一般下载文件都在这个目录
cd /home

下载mongodb

wget https://fastdl.mongodb.org/linux/mongodb-linux-x86_64-4.0.2.tgz

解压mongodb

tar -zxvf mongodb-linux-x86_64-4.0.2.tgz

重命名解压的文件

mv mongodb-linux-x86_64-4.0.2 mongodb

将安装包移动到你需要安装的所在目录

mv mongodb /usr/local/

进入安装目录

cd /usr/local/mongodb

创建数据存放目录

mkdir -p data/db

创建日志存放目录

mkdir logs

进入日志目录

cd logs

创建日志文件

touch mongodb.log

回到/进入mongodb的（启动文件）bin目录

cd /usr/local/mongodb/bin

创建配置文件

vim mongodb.conf

mongodb.conf 配置如下

#auth=true

#bind_ip=0.0.0.0

配置文件保存成功后安装完成

二、测试

运行mongodb（指定配置文件方式运行）
/usr/local/mongodb/bin/mongod -f /usr/local/mongodb/bin/mongodb.conf

连接 mongodb

/usr/local/mongodb/bin/mongo 127.0.0.1:18000

显示如下信息则mongo连接成功

三、创建管理员用户：root

刚安装的mongodb是没有用户及安全设置的，这时我们需要创建一个管理员用户。 步骤如下：连接数据库，use admin数据库，然后创建root用户这三步。

/usr/local/mongodb/bin/mongo 127.0.0.1:18000
use admin
db.createUser({user:”root”,pwd:”admin”,roles:[{role:”root”,db:”admin”}]})

创建完成后 ctrl + c 退出连接

然后修改 mongodb.conf 配置文件

vim /usr/local/mongodb/bin/mongodb.conf

把auth=true注释打开，如下

#bind_ip=0.0.0.0

接下来查看mongodb进程

ps -aux | grep mongod

如下

然后 结束进程并重启mongodb服务

kill -9 8593
/usr/local/mongodb/bin/mongod -f /usr/local/mongodb/bin/mongodb.conf

再次连接 mongodb

/usr/local/mongodb/bin/mongo 127.0.0.1：18000

验证管理员用户，如下，返回1则代表用户验证登陆成功

四、设置远程连接

修改 mongodb.conf 配置文件

vim /usr/local/mongodb/bin/mongodb.conf

把 bind_ip=0.0.0.0 注释打开

保存配置文件后重启一下：分以下3步

启动成功则代表远程访问开启，接下来可以用远程工具输入 ip:port, 用户名，密码 连接 mongodb 了

五、配置开机启动

这里我用的是加一条shell命令进 /etc/bashrc 文件里面，这样系统一开机加载这个文件就会启动mongodb服务了。 在/etc/bashrc文件末尾添加 mongodb 启动命令，如下

mongodb

/usr/local/mongodb/bin/mongod -f /usr/local/mongodb/bin/mongodb.conf

###保存退出后，更新一下此文件
source /etc/bashrc

###至此，mongodb配置完毕，包括（绑定指定端口，创建超级用户及配置验证权限连接，设置远程访问）
六、附：

1、mongodb的启动，登陆和关闭

关闭mongodb：

ps -aux | grep mongod
kill -2 pid

登陆mongodb：

/usr/local/mongodb/bin/mongo 127.0.0.1:18000

2、修改root密码

#auth=true

启动mongodb：

/usr/local/mongodb/bin/mongod -f /usr/local/mongodb/bin/mongodb.conf

登陆mongodb：

/usr/local/mongodb/bin/mongo 127.0.0.1:18000

查看当前所有账户

db.system.users.find()

###删除所有用户
db.system.users.remove({})

创建新的管理员帐号

use admin
db.createUser({user:”root”,pwd:”hyht123”,roles:[{“role”:”root”,”db”:”admin”}]})

重启服务：

use admin
db.auth(“root”,”hyht123”)
db.shutdownServer() 该命令要在root管理员权限下执行
